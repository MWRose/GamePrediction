NFL Outcomes

Description

This project is to create a outcome predictor for NFL football games. This would be done by pulling data from previous years, previous games, or specific game day data. Ideally, this would be more than just wins and losses but also estimated score. One of the goals for this project would be to bring in more information than just the basic stats

Prior data might be

  • Win loss record
  • Historical record against the team
  • Home/away record
  • Season run/pass yards (or allowed)

There is also the oppurtunity to pull in non NFL specific game day data such as

  • Weather
  • Twitter sentiments
  • Time of day

but some of these would take considerable time and work to consolidate. If we did bring this information, probably a lot of the effort would be compiling a new dataset that includes the specific data for each game.

One challenge is that while a random program would only guess right 50% of the time, many programs could probably guess better than this just using a decision tree of wins and losses or other similar statistics. This is why I believe it would be interesting to introduce betting line data for each matchup, that may already have a lot of this information built in. Then instead of only predicting wins and losses, the nueral network would seek to maximize profits. However, this introduces some ethical concerns around gambling and using a computer to do so, so this project would include a consideration of that.

Goals

  • Predict games correctly more than 50% of the time
  • Use NNs to analyze betting lines and payouts to make a profit
  • Explore the ethics of creating a system that can profit by betting
  • Include additional information outside of just sports statistics
